# Nuxt Typescript Template

## Get Started

```bash
git clone https://github.com/seonglae/nuxt-ts-template
yarn
yarn dev
```

## Components

- Nuxt
- Typescript
- Prettier
- Vuetify
- Vuex
- Axios
- SEO Settings
- Nuxt Configs
- Error Page
